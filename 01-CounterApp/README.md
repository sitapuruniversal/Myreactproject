## Create a React + Vite Project

''
npm create vite@latest
''

# type a project name and select Package name

''
Project name 01-CounterApp
Package name 01-COunterApp
''

# Select Framework

''
Select Framework: React
''

# Select Varient

''
Select Varient : Javascript  
''

# Move to the Project

''
cd 01-CounterApp
''

# Install dependency

''
npm install
''

# Run to the Project

''
npm run dev
''
